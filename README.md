# Data-Acquisition
This repo was made in an effrot to find and compare books based on a the weekly listings of a popular review site [thestorygraph.com](https://app.thestorygraph.com/browse). Using the data from this daily updated list you can compare based on ratings, length, or if it's part of a series to help info what you want to read, our help writers learn what types of books are trending.

**booklistScrapper.ipynb:**  This is the scrapper to get the data from the current "Popular This Week List". The data you gain will change week to week.

**popularBooksList.csv** CSV table genertaed the last time I ran the code on 3/18/202
